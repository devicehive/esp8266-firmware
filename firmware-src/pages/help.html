Welcome to DeviceHive ESP8266 firmware!

This firmware allows to connect different devices to DeviceHive cloud service or use local tiny HTTP server with RESTful API embedded rigth into the chip. This page is hosted by that server.

Plenty of different devices can be connected with this firmware. There are few digital interfaces: GPIO, PWM, I2C, SPI and onewire which can be used for connecting device with such interface to IoT cloud. There is also ADC converted on board of this chip, so it's possible to measure voltage of analog sensor. This firmware gives transparent gateway between this physical interfaces and clouds. There are also implementations for some popular devices like DS18B20, DHT22 etc. Please follow the documentaion for details.

Not only DeviceHive cloud service is supported. This firmware provides RESTful API which can be used in local network. Any 3rd party services can send direct HTTP requests on the chip. This endpoint is available at an URL like http://chip-device-id.local/api. It's here for this chip. Firmware announce itself with mDNS, so device can be discovered with local domain name.


Samples of using RESTful API.

Requests can be sent from anywhere from any different services. The most simple and easy way to send such request is JavaScript. Samples below are the simple HTML pages with short JavaScript which perform HTTP requests and display human readable data. In all samples keep AccessKey field empty if it is not configured. Entered AccessKey is saved by browser. Feel free to view page sources directly from there as the examples of using it.

  • Try out RESTful API in your browser tryapi.html
  • GPIO pin handling gpio.html
  • PWM controlling pwm.html
  • Reading analog to digital converter(ADC) voltage adc.html
  • Reading temperature from DS18B20 sensor ds18b20.html
  • Getting humidity from DHT11/DHT22 sensor dht.html
  • Draw an image with SSD1306 OLED display ssd1306.html
  • Print a text on LCD1602 with PCF8574 adapter hd44780.html

Upload web page to chip.

The firmware allows to upload any custom web page which replaces the main page. This page is always available as help.html. It's totally possible to use RESTful API from embedded JavaScript. This feature allows to build web page enabled devices. Samples above can be used as reference.

There is an editor at editor.html page which can be user to edit page rigth in the web browser or upload from file/backup to file.
